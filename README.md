# Student Performance Analytics System (SQL Project)

## Overview
This project analyzes student academic performance using SQL. It demonstrates relational database design, joins, aggregations, and analytical queries.

## Database Structure
- students: stores student details
- subjects: stores subject information
- marks: stores student marks per subject

## Key SQL Concepts Used
- CREATE TABLE
- INSERT INTO
- INNER JOIN
- GROUP BY & HAVING
- Aggregate functions (AVG, SUM, COUNT)
- CASE statements

## Sample Analysis
- Average marks per student
- Top scoring student
- Subject-wise average marks
- City-wise performance
- Identification of low-performing students

## Tools Used
- MySQL 8.0
- MySQL Workbench

## How to Run
1. Run `schema.sql`
2. Run `data.sql`
3. Run `analysis.sql`
